1. `extract_audio_energy(video_path: str) -> list[dict]`
   - Use FFmpeg astats filter: `ffmpeg -i <video> -vn -af "astats=metadata=1:reset=1,ametadata=print:key=lavfi.astats.Overall.RMS_level:file=-" -f null -`
   - Parse stdout for RMS levels per second
   - Returns: `[{time: float, rms_db: float}, ...]`
   - Fallback to volumedetect if astats returns no data

2. `compute_energy_scores(energy_data: list[dict], segments: list[dict]) -> list[float]`
   - Filter out silence (rms_db < -60)
   - Compute mean and std_dev of valid rms values
   - Z-score: z_avg * 0.4 + z_peak * 0.6
   - Normalize to 0-10 range
   - Returns [score, ...] per segment

3. `get_energy_profile(video_path: str, segments: list[dict]) -> dict`
   - Full pipeline: extract + score + find peaks
   - Returns: {energy_data, segment_scores, mean_rms, peak_times}
   - peak_times = top 10% loudest moments

4. `save_energy_data(video_id: str, energy_data: list[dict], segment_scores: list[float], peak_times: list[float]) -> str`
   - Saves to /tmp/clipwise/<video_id>/energy.json
   - Returns path to saved file

```python
from services.energy import get_energy_profile, save_energy_data

class EnergyRequest(BaseModel):
    file_id: str
    segments: list[dict] = []  # optional pre-segmented data

@app.post("/energy")
async def extract_energy(req: EnergyRequest):
    """Extract audio energy from a video file."""
    # Get file path from storage metadata
    info = get_video_info(req.file_id)
    if not info:
        raise HTTPException(status_code=404, detail="Video not found")
    
    video_path = info["path"]
    energy_profile = get_energy_profile(video_path, req.segments)
    
    # Save to /tmp/clipwise/<file_id>/energy.json
    energy_path = save_energy_data(req.file_id, energy_profile["energy_data"], 
                                   energy_profile["segment_scores"], energy_profile["peak_times"])
    
    return {
        "file_id": req.file_id,
        "energy_path": energy_path,
        "peak_times": energy_profile["peak_times"],
        "mean_rms": energy_profile["mean_rms"],
        "segment_scores": energy_profile["segment_scores"]
    }
```
