## Component Inventory

### MomentCard
- States: default, hover (slight border highlight), selected (blue border + checkbox filled)
- Contains: rank number, timestamps, duration, energy score badge, transcript snippet, content type badge, copy button, checkbox
- Energy score badge: green (>70), yellow (40-70), red (<40)

### VideoPlayer
- HTML5 video element at top of dashboard
- Clicking MomentCard jumps video to timestamp via `currentTime`
- Controls: play/pause, progress bar, volume, fullscreen

### TopNSlider
- Range: 3-20
- Default: 10
- Label updates dynamically: "Top {value} momentos"
- Styled with accent color for filled track

### ExportButtons
- Primary: "Enviar pro Opus Clip" (blue, full width at bottom)
- Secondary: "Copiar timestamps" (outline variant)
- Tertiary: "Exportar JSON" (ghost variant)

### TimestampBadge
- Format: `HH:MM:SS - HH:MM:SS`
- Mono font for alignment
- Copy icon on hover
